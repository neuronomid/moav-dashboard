Show the current repository status.

Steps:
1. Display the current branch: `git branch --show-current`
2. Show full status: `git status`
3. Show ahead/behind status compared to remote: `git status -sb`
4. Summarize clearly:
   - Current branch name
   - Number of staged files
   - Number of unstaged/modified files
   - Number of untracked files
   - Whether the branch is ahead/behind the remote
